# Copy tone guide

## Purpose
Tone adjectives from the Brand Strategist are strategic direction. This skill turns them into concrete rules the Copywriter can apply word by word.

## Process

### Step 1: Translate each tone adjective into writing rules

For each adjective from the brand territory, define:
- What it means in sentences (rhythm, length, structure)
- What it means for word choice (vocabulary level, avoid list)
- One example line that embodies it
- One line that violates it

Template:
```
**[Adjective]**
Means: [what this looks like in actual sentences]
Word choice: [concrete vocabulary guidance]
Do: "[example sentence]"
Don't: "[counter-example]"
```

### Step 2: Produce the copy

Apply the tone guide to every deliverable in the brief. Format output as:

```
## Copy deliverables — [Project name]

### Headline (primary)
[Headline]

### Headline variants (3 alternatives)
1. [Variant A]
2. [Variant B]
3. [Variant C]

### Subheadline
[Subheadline]

### Body copy
[Full body text]

### CTA
[Primary CTA text]
[Secondary CTA text if needed]

### Caption (short form, 150 chars max)
[Caption]

### Caption variants by platform
- Instagram: [caption, max 2200 chars, 3-5 hashtags]
- LinkedIn: [caption, professional tone]
- X: [caption, max 280 chars]
```

## Rules
- Every headline must be testable against the single key message. If you remove the headline and replace it with the key message, does the ad still work? If yes, the headline isn't doing enough.
- Never use the word "innovative," "seamless," "powerful," "robust," or "solution" — these are non-words.
- Every CTA must be a verb + benefit. "Start free" not "Click here." "See how it works" not "Learn more."
- Write at least 3 headline variants. The first idea is rarely the best idea.
